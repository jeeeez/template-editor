# @template-editor/react
基于 `@template-editor/native` React 组件的实现。


## 安装
```bash
npm i @template-editor/react
```

## 使用
```typescript
import { ReactTemplateEditor } from '@template-editor/react';


export function Demo(){

  return (
    <ReactTemplateEditor
      value=''
      placeholders={[]}
    />
  );
}
```
